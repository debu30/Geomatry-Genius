The Answers of the questions

1. What is DOM and what is the purpose of DOM?

Answer: DOM refers to Document Object Model. It is represents the objects that includes the structure and componenets of a document on the web. The purpose of DOM is to determine the contents of the webpage and determaine teh how they will be connected or related with each other.


2. How will you select HTML elements using DOM? Name the DOM methods.

Answer: To select HTML element using DOM we need getElementsByTagName() , getElementsByClassName(), getElementById(), querySelector(), querySelectorAll() etc. Some of the DOM methods are using document.getElementsByTagName(), document.getElementsByClassName(), document.getElementById(), document.querySelector(), document.querySelectorAll() etc.


3. What is event bubble?

Answer: The event bubble is an idea when any event haapens in html like click, hove, scroll etc. a group of code or Javascript code executes.


4. What is a callback function and why will you use it?

Answer: A callback function is a type of function that assigns objects with function or any type of code that executes when the function is called. It pass functions as parameters to other functions and call them inside the outer functions.
